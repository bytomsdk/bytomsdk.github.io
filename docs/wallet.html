backup-wallet

备份钱包到镜像文件，它包含帐户镜像，资产镜像和密钥镜像。

参数

none

返回

  • Object - account_image, 账户镜像

例子

Client client = TestUtils.generateClient();
Wallet wallet = Wallet.backupWallet(client);

restore-wallet

按镜像文件还原钱包。

参数

  • Object - account_image, 账户镜像
  • Object - asset_image, 资产镜像
  • Object - key_images, 密钥镜像

返回

如果恢复钱包成功，返回none

例子

Client client = TestUtils.generateClient();
Wallet wallet = Wallet.backupWallet(client);
Wallet.restoreWallet(client, wallet.accountImage, wallet.assetImage, wallet.keyImages);

// Result
//none
